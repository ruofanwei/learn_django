### Installation & Setup

- `pip install - pip3 install package_name --user`

##### pip的基本功能 - 安裝套件
- `python3.9 -m pip install --upgrade pip`

##### pip的基本功能就是安裝套件
- `pip3 install virtualenv`
##### 安裝 virtualenv
- `python3 -m virtualenv env` 
##### 進入虛擬環境模式
- `source ./env/bin/activate`

- `pip3 install Django`


### Run project
- `python3 manage.py runserver`
